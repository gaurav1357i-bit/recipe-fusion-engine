# What's for Dinner - Recipe Fusion Engine 🍳🔥

An interactive, single-page web application that helps users discover unique culinary combinations through an intelligent Recipe Fusion Engine. It features a modern dark-themed UI, an admin dashboard, and an automated nutrition calculator.

## 🌟 Key Features

* **Dual Authentication System:** Separate login and registration flows for Users and Admins.
* **Intelligent Recipe Fusion:** Mix and match categories and cuisines to generate unique recipe suggestions and multi-cuisine mashups.
* **Dynamic Nutrition Calculator:** Automatically estimates calories, protein, carbs, and fats based on the exact ingredients used in a recipe.
* **Subscription Management:** Built-in tier system featuring a Free plan (limited daily recipes) and Premium plans (unlimited access).
* **Admin Dashboard:** Fully functional admin panel to add, delete, and manage recipes, categories, cuisines, and broadcast messages to users.
* **Local Database:** All user data, recipes, and preferences are seamlessly persisted using the browser's `localStorage`.
* **Responsive & Modern UI:** Clean interface built with CSS variables, featuring smooth transitions, custom dropdowns, and lazy-loaded images.

## 🛠️ Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (Flexbox, CSS Grid, Custom Variables)
* **Logic:** Vanilla JavaScript (ES6+, Classes, Maps/Sets)
* **Icons & Typography:** FontAwesome & Google Fonts (Roboto)
* **Storage:** Browser `localStorage` (No external backend or database required)
